Frequently asked
How do I enable Spotify? ▼
Set SPOTIFY_CLIENT_ID & SPOTIFY_CLIENT_SECRET on the server, restart Flask and call /api/debug_spotify to verify.
YouTube API not working? ▼
Ensure YOUTUBE_API_KEY is set and has YouTube Data API v3 access. Use /api/debug_youtube_auto for diagnostics.
Is my camera data stored? ▼
No — face & voice analysis runs in the browser by default; only mood labels are optionally posted (no raw audio/video).
